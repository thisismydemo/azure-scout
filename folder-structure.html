Repository Structure ​

Overview ​

This page documents the repository layout after the Phase 1B reorganization. Numbered-prefix folders were renamed, duplicate directories merged, legacy code removed, and the docs tree uses MkDocs Material with Markdown.

Module Loading ​

The PSM1 loader uses Get-ChildItem -Recurse *.ps1. Folder names are cosmetic — renaming or merging directories has zero impact on which functions are loaded. 13 public functions are exported via the PSD1 manifest.
