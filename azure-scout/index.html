Azure Tenant Inventory

Overview

Azure Tenant Inventory (AZSC) is a PowerShell module that generates comprehensive inventory reports for a single Azure tenant — covering both ARM resources and Entra ID (Azure AD) objects.

Feature Description

ARM Resource Inventory

95 resource modules across 17 categories (AI, Analytics, APIs, AzureLocal, Compute, Container, Database, Hybrid, Identity, Integration, IoT, Management, Monitoring, Network, Security, Storage, Web)

Entra ID Inventory

15 identity modules — Users, Groups, Applications, Service Principals, Conditional Access, PIM, Administrative Units, Named Locations, Domains, Authentication Methods, and more

Excel Reports

Rich multi-worksheet spreadsheets with charts and conditional formatting

JSON Output

Machine-readable normalized output for automation pipelines

Network Diagrams

Auto-generated draw.io topology diagrams

Pre-flight Permission Checker

Validates ARM + Graph access before running

Quick Start

# Import from local clone
Import-Module ./AzureScout.psd1

# Run full inventory (uses current Azure context)
Invoke-AzureScout

# ARM-only scan with JSON output
Invoke-AzureScout -Scope ArmOnly -OutputFormat Json

# Entra ID only
Invoke-AzureScout -Scope EntraOnly
If you’re already logged in via Connect-AzAccount, AZSC uses your existing session — no additional flags needed.
