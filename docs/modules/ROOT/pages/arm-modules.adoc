= ARM Inventory Modules
:description: Complete catalog of AzureScout Azure Resource Manager inventory modules across 18 categories.
:keywords: arm, modules, categories, resources, inventory, azurescout

== Overview

AzureScout includes *95 ARM inventory modules* organized into *16 categories*.
Each module extracts data for a specific Azure resource type using the Azure Resource Manager APIs.

Run ARM-only extraction with:

[source,powershell]
----
Invoke-AzureScout -Scope ArmOnly
----
