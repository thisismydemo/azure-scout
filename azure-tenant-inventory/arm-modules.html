ARM Inventory Modules

Overview

Azure Tenant Inventory includes 95 ARM inventory modules organized into 16 categories. Each module extracts data for a specific Azure resource type using the Azure Resource Manager APIs.

Run ARM-only extraction with:

Invoke-AzureTenantInventory -Scope ArmOnly
